Instalação do clamav em servidor Ubuntu/Debian

sudo su
apt-get update
apt-get install clamav clamav-daemon
freshclam

Checando

clamscan -r /home/ribafs
clamscan -r /

Varrer todo o computador

clamscan -r --bell -i /

Criar lista de arquivos infectados

clamscan -r /home/ribafs/ | grep FOUND >> report.txt

Checar Versão

clamdscan -V

Adicionando ao cron

crontab -e

00 00 * * * clamscan -r /home

Instalar gui

apt-get install ClamTK
